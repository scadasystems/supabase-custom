#  Build

- docker build . -f apps/studio/Dockerfile --target production -t studio:latest

## Run

- docker run -p 3000:3000 supabase/studio

## Deploy

- docker push supabase/studio:latest

## Clean build

- docker builder prune
- docker buildx build . -f apps/studio/Dockerfile --target production -t flintapp/studio:20250205 --no-cache

## 멀티 아키텍처 빌드
- docker buildx create --name flintapp
- docker buildx use flintapp
- docker buildx build --platform linux/amd64,linux/arm64 -f apps/studio/Dockerfile --target production -t flintapp/studio:20251007 --push .
